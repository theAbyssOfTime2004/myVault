## Ngày 5 — Sorting + Bit

- [ ] **1h** — Ghi nhớ bảng complexity (tự viết tay không nhìn tài liệu):

| Algorithm | Worst | Ghi chú |
|---|---|---|
| Bubble / Insertion | O(n²) | |
| Merge Sort | O(nlogn) | Stable |
| Quick Sort | **O(n²)** | Worst case = dãy đã sorted |
| Heap Sort | O(nlogn) | In-place |

- [ ] **30p** — Bit ops: trace tay `5&3`, `5|3`, `5^3`, `5<<1`, `5>>1`. Đọc Kadane's algorithm, trace với `[-2,1,-3,4,-1,2,1,-5,4]`
- [ ] **30p** — Làm quiz [Sorting GQ](https://www.geeksforgeeks.org/algorithms-gq/sorting-gq/)
