# Figures

Two kinds of figures in the thesis:

- **Conceptual / flow diagrams** — written as `mermaid` blocks inline in the chapter Markdown (render in Obsidian; convert to TikZ/SVG at the LaTeX stage). These need no script.
- **Data plots** — generated by `make_figures.py` (matplotlib, APA 7.0 styling, Tol colorblind-safe palette).

## Running

```bash
cd figures
python make_figures.py        # needs matplotlib + numpy
```

Outputs `fig_4_1 .. fig_4_7` as both `.pdf` (for LaTeX) and `.png` (for the Obsidian/Markdown preview). The chapters embed the `.png` paths, so the figures appear in the Markdown once the script has been run. Run it in the same environment used for the experiments (Colab/WSL), since Python is not installed on the report machine.
