### Implement Stack Using Queues

```python
class MyStack(object):
	from collections import deque
	def __init__():
		self.q = deque()
	
	def push(self, x):
		self.q.append(x)
		for _ in range(len(self.q) - 1):
				self.q.append(self.q.popleft())
	
	def pop(self):
		return self.q.popleft()
```
